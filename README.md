# Student Report Card (Lab 3)

A React application built for Web Dev II Lab Assignment 3. It uses a component-based architecture to display, add, and dynamically update student scores while automatically calculating pass/fail status.

## Features

* **Student Table:** Displays a list of students, their scores, and current status.
* **Add New Students:** A form to input student names and marks.
* **Dynamic Scoring:** Update existing scores directly in the UI.
* **Status Calculation:** Automatically evaluates and color-codes pass/fail status (Pass is >= 40).

## Tech Stack

* React (Vite)
* Tailwind CSS
* JavaScript (JSX)
